# 🎙️ Voice-Controlled AI Agent

## 🚀 Overview
This project is a local AI agent that accepts audio input, converts it into text, detects user intent, executes corresponding actions, and displays results via a web UI.

## ⚙️ Features
- Audio input (.wav/.mp3)
- Speech-to-Text using Whisper
- Intent Detection
- Actions:
  - Create files
  - Write code
  - Summarize text
  - Chat
- Streamlit UI

## 🧠 Architecture
Audio → Whisper → Text → Intent Detection → Action Execution → UI Output

## 🛠️ Tech Stack
- Python
- Streamlit
- Whisper (Speech-to-Text)
- File System Operations
